# 📖 About This Portfolio

Welcome to my **Data Analytics Internship Portfolio**, developed during my internship at **ApexPlanet Software Pvt. Ltd.**

This portfolio showcases my complete journey through the internship, covering the entire data analytics lifecycle—from data cleaning and preprocessing to exploratory data analysis, SQL-based business intelligence, interactive dashboard development, data storytelling, and statistical validation.
